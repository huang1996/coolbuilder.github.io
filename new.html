H5学习记录

什么是HTML5？

按照W3C给定的标准：HTML5 将成为 HTML、XHTML 以及 HTML DOM 的新标准。
其实简单点说就是HTML第五个版本

HTML5有什么新特性？

HTML5带来了很多新东西，也对之前的版本的一些东西进行了优化，还规范了代码书写标准，总的来说，是一次很大的进步。

H5的几个重要功能

Canvas

HTML5 的 canvas 元素使用 JavaScript 在网页上绘制图像。 Canvas_demo

SVG

HTML5 的 canvas 元素使用 JavaScript 在网页上绘制图像。

地理定位

HTML5 Geolocation API 用于获得用户的地理位置。

Web存储

HTML5 提供了两种在客户端存储数据的新方法：
localStorage - 没有时间限制的数据存储
sessionStorage - 针对一个 session 的数据存储

应用缓存

HTML5 引入了应用程序缓存，这意味着 web 应用可进行缓存，并可在没有因特网连接时进行访问。

服务器发送事件

Server-Sent 事件指的是网页自动获取来自服务器的更新。
